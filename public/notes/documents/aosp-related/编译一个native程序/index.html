编译一个native程序

在Android Studio中编译

  1. 创建一个Native C++ Project
  2. 寻找到Android Studio自动创建的CMakeLists.txt(一般在项目的build.gradle.kts中会指定此CMakeLists.txt)
  3. 编辑源码与CMakeLists.txt
  4. 使用Android Studio编译即可, 最后的产物会在app/build文件夹下

单独使用NDK编译

另一种比较直观的方法就是使用NDK单独编译, 其实上文中描述的在Android Studio中编译也是利用了NDK.

NDK中包含了跨平台工具链, 在ndk/toolchains/llvm/prebuilt/linux-x86_64中, 使用此工具链可以完成编译.

cmake -DCMAKE_TOOLCHAIN_FILE=$NDK/build/cmake/android.toolchain.cmake -DANDROID_ABI=$ABI

具体编译过程可见丢帧预测 注: 有些项目提供的aar包解压后就是头文件, 动态库.
