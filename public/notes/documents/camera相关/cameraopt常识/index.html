cameraopt常识

cameraopt_release的分支要在SSI中查. 编译出的产物位置:out/target/product/missi/system_ext/framework/miui-cameraopt.jar cameraopt_release位置:release/miui-cameraopt/libs/miui-cameraopt.jar 手机系统中位置:system_ext/framework/miui-cameraopt.jar

cameraopt log开关

adb root
adb remount
adb shell setprop persist.vendor.cameraopt.loglevel 1
adb shell setprop persist.sys.lmkd.killing.debug true
adb shell setprop persist.sys.miui.cameraopt.debug true
adb shell setprop persist.sys.miui.camera.boost.debug true
adb shell pkill sys
