Log开关

Log开关

可以用

adb shell stop logd
adb shell start logd

来开关logd, 排除log的影响, 用trace来看性能

关log

adb root adb shell setprop vendor.debug.camera.ulog.mode 0x10 adb shell setprop persist.vendor.mtk.camera.log_level -1 adb shell setprop persist.sys.offlinelog.logcat false adb shell setprop persist.sys.offlinelog.logcatkernel false adb shell setprop persist.vendor.camera.mivi.groupsEnable 0 adb shell setprop persist.vendor.camera.offlinedebug.mask 0 adb shell setprop persist.vendor.camera.mialgo.tracegroups 0 adb shell setprop camera.debug.log_level 4 adb shell pkill camera*

adb wait-for-device
adb root
#adb disable-verity
adb remount
#adb reboot
adb wait-for-device

#close log by set persist
adb shell setprop persist.vendor.camera.debugdata.enable3A 0
adb shell setprop persist.vendor.camera.debugdata.enableTuningMetadata 0

adb shell setprop persist.vendor.camera.logInfoMask 0
adb shell setprop persist.vendor.camera.logVerboseMask 0
adb shell setprop persist.vendor.camera.logConfigMask 0
adb shell setprop persist.vendor.camera.logWarningMask 0
adb shell setprop persist.vendor.camera.systemLogEnable 0
adb shell setprop persist.vendor.camera.logCoreCfgMask 0
adb shell setprop persist.vendor.camera.traceGroupsEnable 0
adb shell setprop vendor.debug.camera.overrideLogLevels 0x1

adb shell setprop persist.vendor.camera.enableAsciiLogging 0
adb shell setprop persist.vendor.camera.enableTxtLogging 0
adb shell setprop persist.vendor.camera.raiserecoverysigabrt FALSE
adb shell setprop persist.vendor.camera.enableCameraCoreDumpBinary FALSE
adb shell setprop persist.vendor.camera.enableCameraCoreDumpText FALSE

adb shell setprop persist.vendor.camera.logRequestMapping FALSE

#closeAsdLog
adb shell setprop persist.vendor.camera.mawlog.aiasd 0
adb shell setprop persist.vendor.camera.mawtools.offlinelog 0
adb shell setprop persist.vendor.camera.mawlog 0

#closeBspOfflineLog
adb shell setprop persist.sys.offlinelog.logcat false
adb shell setprop persist.sys.offlinelog.logcatkernel false

#remove ssgtzd
adb shell "rm /vendor/bin/ssgtzd"

#close mivi log
adb shell setprop persist.vendor.camera.mivi.groupsEnable 0
adb shell setprop persist.vendor.camera.offlinedebug.mask 0
adb shell setprop persist.vendor.camera.mialgo.tracegroups 0

#close MiuiCamera debug log
#adb shell setprop camera.debug.log_level 4
adb shell setprop camera_dump_parameters true
adb shell setprop camera.debug.log_level 1
adb shell setprop camera.debug.dump_stat_event true

adb shell pkill camera*
