Profiling with perf

我们可以运用perf来对我们的程序进行profiling, 其原理类似于Android的Simpleperf(倒反天罡).

quick use

# -g for call graph
# -F for frequency
perf record -g ${your_excutable}
# 可以使用默认的perf report
perf report
# 但是更推荐另一种方法, 生成火焰图, 更加直观
# 需要FlameGraph(https://github.com/brendangregg/FlameGraph)
perf script | ./stackcollapse-perf.pl | ./flamegraph.pl > perf.svg
# open perf.svg with your broswer

评价是不如simpleperf提供的脚本, 可以参考Simpleperf简明教程, 使用这里生成的perf.data, 直接使用report_html.py生成报告.
