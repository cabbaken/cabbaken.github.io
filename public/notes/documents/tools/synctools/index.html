ZeroTier

多端互连的基础

LocalSend

LocalSend支持多端, 缺点是只能识别一个网络. 建议与ZeroTier搭配使用.

clipboard-online

clipboard-online支持多端, 可以自己写一个接受器和发送器, 接口简单. 具体可见下文代码.

import requests

URL = "http://192.168.194.60:8086"
HEADER={'X-API-Version':'1'}

res = requests.get(url=URL,headers=HEADER)
print(res.json())
