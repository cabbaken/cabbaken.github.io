编译为native程序

注: 本文使用onnxruntime-1.20.0 Android不带python环境, 所以需要把模型打包为onnx格式, 编译一个适配的native程序调用模型. 需要手写一个CMakeLists.txt利用NDK进行编译

cmake_minimum_required(VERSION 3.10)

project(onnx_model)

set(CMAKE_EXPORT_COMPILE_COMMANDS ON)

add_executable(load load.cpp)

target_include_directories(load PUBLIC onnxruntime/onnxruntime-android-1.20.0/include)

target_link_directories(load PUBLIC onnxruntime/onnxruntime-android-1.20.0/lib)

target_link_libraries(load onnxruntime)

target_compile_options(load PUBLIC -g)

这里有一个很坑的点, microsoft github release中并没有提供onnx在android平台的库, 需要从这里下载aar文件然后解压得到. 这个方法来自: https://onnxruntime.ai/docs/install/#install-on-android 然后

# 这里是为了利用NDK编译, 使用NDK提供的cmake脚本设置一些变量和环境
cmake -DCMAKE_TOOLCHAIN_FILE=$NDK/build/cmake/android.toolchain.cmake -DANDROID_ABI=$ABI
make

运行前需要把使用的动态库也放到设备的对应库的路径中.

前景

后期可能需要间隔运行, 如隔3s/10s运行一次, 确保性能. 在运行的时候实时读取当前的信息, 根据信息预测未来的丢帧情况.

可以集成在CameraBoost, MIVI框架中, 让其对未来状况有感知. 一个典型场景: 高负载时, 传统的方案无法有效进行处理, 导致大量丢帧情况, 用户感知明显卡顿. 此模型可以让CameraBoost获取未来帧率信息, 根据不同的等级具体处理丢帧情况, 优化系统CPU和内存状态, 提升相机使用体验.

    public static boolean dataFromFileKeyword(String filePath,
        List<String> keyWords,
        Map<String, String> data) {
        if (filePath == null) {
            System.out.println("qiuruoyu filePath is null");
            return false;
        }

        File file = new File(filePath);
        if (!file.exists()) {
            System.out.println("qiuruoyu Failed to open file path: " + filePath);
            return false;
        }

        try (BufferedReader reader = new BufferedReader(new FileReader(file))) {
            String line;
            while ((line = reader.readLine()) != null) {
                if (keyWords.isEmpty()) {
                    break;
                }
                Iterator<String> iterator = keyWords.iterator();
                while (iterator.hasNext()) {
                    String keyword = iterator.next();
                    if (!line.contains(keyword)) {
                        continue;
                    } else {
                        String[] parts = line.split(":");
                        if (parts.length > 1) {
                            data.put(keyword, parts[1].trim());
                            // iterator.remove();
                        }
                        break;
                    }
                }
            }
        } catch (Exception e) {
            System.out.println("qiuruoyu Error dataFromFileKeyword(): " + e.getMessage());
            return false;
        }
        return true;
    }

    public static boolean dataFromPath(String path, int[] data) {
        if (path == null || path.isEmpty()) {
            System.out.println("qiuruoyu Path is null or empty");
            return false;
        }

        try {
            byte[] bytes = Files.readAllBytes(Paths.get(path));
            String content = new String(bytes).trim();

            if (!content.isEmpty()) {
                try {
                    data[0] = Integer.parseInt(content);
                } catch (NumberFormatException e) {
                    System.out.println("qiuruoyu Cannot parse HW status as integer: " + content);
                    return false;
                }
            } else {
                System.out.println("qiuruoyu Cannot get HW status");
            }
            return true;
        } catch (Exception e) {
            System.out.println("qiuruoyu Error dataFromPath(): " + e.getMessage());
            return false;
        }
    }
