使用大模型

使用大模型

由于大模型文件较大, 需要安装git-lfs.

sudo apt install git-lfs

找到一个大模型(一般在huggingface.co或是其他下载站)

根据pytorch配置环境 https://pytorch.org/

(或许会发生以下情况)

conda config --set channel_priority flexible

获取模型后, 可以利用transformers快捷调用模型

from transformers import LlamaForCausalLM, AutoTokenizer

model = LlamaForCausalLM.from_pretrained(".",)

tokenizer = AutoTokenizer.from_pretrained(".",)

inputs = tokenizer("Could you generate some c++ code? Please show me a c++ hello world.", add_special_tokens=False, return_tensors="pt")
tokens = model.generate(**inputs, max_new_tokens=400)

print(tokenizer.decode(tokens[0]))
